# INTRODUCTION TO PYTHON

## 1. PYTHON

- Criado em 1991 por Guido van Rossum
- Linguagem que atende a vários propósitos
- Características da linguagem:
  - Alto nível
  - Open source
  - Multiplataforma
  - Fácil de aprender
  - Amplamente utilizada para área de dados e pesquisas
